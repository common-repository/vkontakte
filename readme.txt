== Description ==

The plugin adds a wide range of VKontakte functionality to your site.
The following feature are currently supported:

* comments
* like button
* share button
* group widget
* poll widget
* recommendations widget

== Installation ==

For the plugin to work, you need to specify the VKontakte application ID. You can get it [here](https://vk.com/editapp?act=create).
